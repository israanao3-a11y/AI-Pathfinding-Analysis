# AI Pathfinding Analysis Tool 🚀

## Description (FR)
Ce projet est une étude comparative des performances de quatre algorithmes de recherche de chemin (Pathfinding) : **A***, **BFS**, **DFS**, et **Greedy Best-First Search (GBFS)**. L'objectif est d'analyser l'efficacité de chaque algorithme en termes de temps d'exécution et de consommation de mémoire dans un labyrinthe donné.

## Features
* **Comparaison multidimensionnelle :** Analyse du temps, de la mémoire et de la longueur du chemin.
* **Visualisation Data :** Génération automatique de graphiques radars (Radar Charts) avec Matplotlib.
* **Statistiques :** Utilisation de Pandas pour calculer la moyenne et l'écart-type des performances.
